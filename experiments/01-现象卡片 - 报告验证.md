## 规格定义（形式化）

### 报告不变式

```json
{
  "type": "object",
  "required": ["title", "date", "content", "sources"],
  "properties": {
    "title": {"type": "string", "minLength": 10},
    "date": {"type": "string", "format": "date"},
    "content": {"type": "string", "minLength": 100},
    "sources": {
      "type": "array",
      "minItems": 5,
      "items": {
        "type": "object",
        "required": ["url", "publish_date"],
        "properties": {
          "url": {"type": "string", "format": "uri"},
          "publish_date": {"type": "string", "format": "date"}
        }
      }
    }
  }
}
```

### 验证规则

| 规则 ID | 名称 | 检查项 | 阈值 |
|--------|------|--------|------|
| V1 | 非空测试 | 正文字数 | ≥ 100 |
| V2 | 引用数量 | sources 数量 | ≥ 5 |
| V3 | 章节数量 | Markdown 章节数 | ≥ 3 |
| V4 | 引用完整 | 每个来源有 URL+ 日期 | 100% |
| V5 | 格式正确 | Markdown 解析 | 无错误 |
| V6 | 时间一致 | 报告日期 = 生成日期 | 匹配 |
